\# Student Task \& Progress Tracker



\## Overview

A full-stack web application built using Flask (Python), HTML, and CSS to help students manage tasks, track deadlines, and monitor overall progress.



\## Features

\- User authentication (login/signup)

\- Task management system (priority, deadlines, descriptions)

\- Progress tracking dashboard

\- Dynamic completion bar



\## Tech Stack

\- Python (Flask)

\- HTML

\- CSS



\## Status

Currently under active development.
